Region Finder & Pollen API Documentation

Welcome to the pollen API proxy. Use this documentation to understand how to interact with the API. The Pollen data is provided by Norges Astma- og Allergiforbund

Endpoints

GET /find_region

Returns the region for the given latitude and longitude. Useful for mapping pollen regions to municipalities.

Parameters:

  • lat (float): Latitude of the location
  • lon (float): Longitude of the location

Example Request:

GET /find_region?lat=59.9139&lon=10.7522

Example Response:

{
    "kommune": "Oslo",
    "pollen_region": "ostlandetMedOslo"
}

GET /pollen

Returns the pollen data for the given latitude and longitude.

Parameters:

  • lat (float): Latitude of the location
  • lon (float): Longitude of the location

Example Request:

GET /pollen?lat=59.9139&lon=10.7522

Example Response:

{
    "kommune": "Askøy",
    "pollen_data": [
        {
            "displayName": "Hordaland",
            "id": "hordaland",
            "textForecast": "Bjørkepollensesongen er nå i gang over hele regionen. Nedbør kan hemme spredningen midlertidig."
        }
    ]
}
