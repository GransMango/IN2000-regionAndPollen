Region Finder API Documentation

Welcome to the Region Finder API. Use this API to determine the region based on latitude and longitude.

Endpoints

GET /find_region

Returns the region for the given latitude and longitude.

Parameters:

  • lat (float): Latitude of the location
  • lon (float): Longitude of the location

Example Request:

GET /find_region?lat=59.9139&lon=10.7522

Example Response:

{
    "region": "Oslo"
}
