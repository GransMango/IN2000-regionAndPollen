Region Finder & Pollen API Documentation

Welcome to the pollen api proxy. The pollen data is provided by NAAF (Norges Asthma og Allergi forbund)

Endpoints

GET /find_region

Returns the region for the given latitude and longitude. Useful if you want to implement your own mapping from pollen regions to municipalities

Parameters:

  • lat (float): Latitude of the location
  • lon (float): Longitude of the location

Example Request:

GET /find_region?lat=59.9139&lon=10.7522

Example Response:

{
    "region": "Oslo"
}

GET /pollen

Returns the pollen data for the given latitude and longitude.

Parameters:

  • lat (float): Latitude of the location
  • lon (float): Longitude of the location

Example Request:

GET /pollen?lat=59.9139&lon=10.7522

Example Response:

{
    "region": "Oslo",
    "pollen_data": [ ... ]
}

GET /pollen/regions

Returns the list of pollen regions.

Example Request:

GET /pollen/regions

Example Response:

[
    {
        "displayName": "Finnmark",
        "id": "finnmark"
    },
    ...
]
